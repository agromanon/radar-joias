# Radar Leilão - Oportunidades Industriais B2B

O **Radar Leilão** é um ecossistema SaaS de alta performance desenhado para fundos de investimento e compradores industriais. Ele automatiza a varredura, análise e precificação de leilões judiciais e extrajudiciais no Brasil, transformando editais complexos em decisões lucrativas.

## 🚀 Principais Funcionalidades

- **Radar Copilot (IA Agentic):** Agente de inteligência artificial exclusivo para o plano *War Room*. Capaz de realizar buscas, configurar alertas e analisar riscos via chat de linguagem natural.
- **Auditoria de Editais:** Engine Multi-LLM que rastreia taxas ocultas, dívidas ativas e restrições judiciais em segundos.
- **Radar Geográfico:** Mapa de calor interativo de zonas de oportunidade com isolamento logístico para o território brasileiro.
- **Tracker Autônomo:** Sistema de Smart Match que notifica oportunidades críticas via Push/Email 24/7.
- **Admin Control Room:** Painel de gestão de infraestrutura, roteamento de modelos de IA e métricas de negócio (MRR).

## 🛠️ Tech Stack

- **Frontend:** Next.js 15 (App Router), React, Tailwind CSS v4.
- **Iconografia:** Lucide React.
- **Mapa:** Leaflet.js com tiles customizados (CartoDB Dark).
- **Backend/DB:** Supabase (Auth, Postgres, Storage).
- **IA:** Integração Multi-Provider (DeepSeek, Qwen) via roteador inteligente.

## 🏁 Início Rápido

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) para ver o portal de marketing ou [/dashboard](http://localhost:3000/dashboard) para o app.
